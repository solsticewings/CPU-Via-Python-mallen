# CPU-Via-Python-mallen
An implementation of a 16-bit Harvard architecture CPU with a limited instruction set.
Done in a group of three, using starter code and a runner by Clayton Cafiero at the University of Vermont.

I completed the ALU and the memory implementation, as well as two assembly programs and most of the instructions in the CPU proper.

Program is run via "run.py" and takes input (a .asm file to open) via command prompt.
